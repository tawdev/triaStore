# Tria Lamp (TriaStor) Project

This is a professional full-stack project structure for TriaStor.

## Structure

- `frontend/`: Next.js application for the store front and admin panel.
- `backend/`: NestJS API application for the store backend.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

From the root directory, run:
```bash
npm install
```

### Running the Project

To run both frontend and backend in development mode:
```bash
npm run dev
```

Or run them individually:
```bash
# Frontend only
npm run dev:frontend

# Backend only
npm run dev:backend
```
